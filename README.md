<h2>Overview</h2>
<p>final-project-stock-manager is a Spring Boot-based application designed to facilitate efficient inventory and sales management with secure role-based access. It supports user authentication, product categorization, and transaction tracking within a structured architecture.</p>
<p><strong>Why final-project-stock-manager?</strong></p>
<p>This project aims to provide a reliable backend system for stock control and user management. The core features include:</p>
<ul class="list-disc pl-4 my-0">
<li class="my-0">🛡️ <strong>Security &amp; Authentication:</strong> Implements JWT tokens and role validation to ensure secure user access.</li>
<li class="my-0">📊 <strong>API Documentation:</strong> Uses OpenAPI for seamless API exploration and integration.</li>
<li class="my-0">🏷️ <strong>Inventory Management:</strong> Manages products, categories, and sales with real-time data validation.</li>
<li class="my-0">🔄 <strong>Data Integrity:</strong> Centralized exception handling and validation annotations maintain data consistency.</li>
<li class="my-0">💾 <strong>Database Integration:</strong> Connects effortlessly with MySQL for persistent storage.</li>
<li class="my-0">💱 <strong>Financial Accuracy:</strong> Retrieves real-time USD exchange rates for precise financial calculations.</li>
</ul>

<h2>Getting Started</h2>
<h3>Prerequisites</h3>
<p>This project requires the following dependencies:</p>
<ul class="list-disc pl-4 my-0">
<li class="my-0"><strong>Programming Language:</strong> Java</li>
<li class="my-0"><strong>Package Manager:</strong> Maven</li>
</ul>
<h3>Installation</h3>
<p>Build final-project-stock-manager from the source and install dependencies:</p>
<ol>
<li class="my-0">
<p><strong>Clone the repository:</strong></p>
<pre><code class="language-sh">❯ git clone https://github.com/santipoido/final-project-stock-manager
</code></pre>
</li>
<li class="my-0">
<p><strong>Navigate to the project directory:</strong></p>
<pre><code class="language-sh">❯ cd final-project-stock-manager
</code></pre>
</li>
<li class="my-0">
<p><strong>Install the dependencies:</strong></p>
</li>
</ol>
<p><strong>Using <a href="https://maven.apache.org/">maven</a>:</strong></p>
<pre><code class="language-sh">❯ mvn install
</code></pre>
<h3>Usage</h3>
<p>Run the project with:</p>
